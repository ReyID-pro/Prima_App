# 📊 Prima_App - School Dashboard System

Sistem aplikasi dashboard sekolah berbasis **Google Apps Script (GAS)** yang ditampilkan melalui **GitHub Pages** menggunakan iframe.

Aplikasi ini dirancang untuk kebutuhan administrasi sekolah, monitoring data, dan akses cepat berbasis web.

## 🎯 Fitur Utama

- Dashboard sekolah berbasis web
- Akses cepat via browser (HP & PC)
- Integrasi Google Apps Script
- Tampilan ringan & responsif
- Bisa diakses melalui GitHub Pages
- Sistem online real-time

---

## 🧩 Teknologi yang Digunakan

- Google Apps Script (Backend & Web App)
- HTML, CSS, JavaScript (Frontend)
- Google Drive (Storage)
- Google Spreadsheet (Database)
- GitHub Pages (Hosting tampilan)
